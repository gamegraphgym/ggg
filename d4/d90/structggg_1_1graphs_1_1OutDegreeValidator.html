template<std::size_t MinOutDegree = 0, std::size_t MaxOutDegree = std::numeric_limits<std::size_t>::max()>
struct ggg::graphs::OutDegreeValidator< MinOutDegree, MaxOutDegree >

Validator for out-degree range of vertices.

This validator checks that all vertices have an out-degree within a specified range [MinOutDegree, MaxOutDegree]. Useful for ensuring game graphs have valid moves and bounded branching factor.

Template Parameters
MinOutDegreeMinimum required out-degree (default: 0, meaning no minimum check)
MaxOutDegreeMaximum allowed out-degree (default: unlimited, meaning no maximum check)

Example usage:

// Require at least one outgoing edge per vertex
OutDegreeValidator<1>::validate(graph);
// Require between 2 and 5 outgoing edges per vertex
OutDegreeValidator<2, 5>::validate(graph);
// Require exactly 3 outgoing edges per vertex
OutDegreeValidator<3, 3>::validate(graph);
// Require at most 10 outgoing edges per vertex (no minimum)
OutDegreeValidator<0, 10>::validate(graph);

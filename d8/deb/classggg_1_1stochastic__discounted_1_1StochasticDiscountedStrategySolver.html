Detailed Description

Strategy Improvement solver for stochastic discounted games.

Implementation of the strategy iteration algorithm for solving stochastic discounted games based on [DBLP:journals/or/Howard60] and [DBLP:journals/corr/DellErbaDS24]. Uses linear programming to solve the value equations at each iteration.

Detailed Description

Priority Promotion (PP) parity game solver.

Implementation of the Priority Promotion algorithm for solving parity games based on [DBLP:journals/fmsd/BenerecettiDM18]. The implementation follows the approach from the Oink solver [DBLP:conf/tacas/Dijk18].

The algorithm iterates vertices from highest to lowest priority, maintaining regions of vertices and promoting them to higher priorities when regions become closed. When a region becomes a dominion, all vertices in it are solved.

Time complexity: O(2^n), Space: O(n)

Detailed Description

Mean-payoff Solver with Constraint Analysis (MSCA)

Implementation of the MSCA algorithm for solving mean-payoff games based on [DBLP:journals/fmsd/BrimCDGR11] and [DBLP:conf/icalp/DorfmanKZ19]. This algorithm uses constraint analysis and scaling techniques for efficient computation.

Detailed Description

template<typename GraphType>
class ggg::solutions::RSolution< GraphType >

Solution that stores per-vertex winning-region information.

Capability: R (Regions)

RSolution provides methods to query and set the winning player for individual vertices and to obtain the full map of winning regions. It also implements JSON/text conversion helpers.

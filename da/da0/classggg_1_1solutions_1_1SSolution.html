Detailed Description

template<typename GraphType, typename StrategyType = ggg::strategy::DeterministicStrategy<GraphType>>
class ggg::solutions::SSolution< GraphType, StrategyType >

Solution that stores a winning strategy.

Capability: S (Strategies)

SSolution provides accessors to set, query and retrieve strategies for vertices. It emits strategy information in JSON/text form using the project's strategy helpers.

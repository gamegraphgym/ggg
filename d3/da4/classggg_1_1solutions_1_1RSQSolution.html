Detailed Description

template<typename GraphType, typename StrategyType = ggg::strategy::DeterministicStrategy<GraphType>, typename ValueType = double>
class ggg::solutions::RSQSolution< GraphType, StrategyType, ValueType >

Convenience solution combining Regions, Strategies and Quantitative values.

Capability: R + S + Q

RSQSolution aggregates RSolution, SSolution and QSolution to provide regions, strategies and per-vertex quantitative values in a single type.

MSE (Mean payoff Solver using Energy games) solver for mean payoff vertex games.

Implementation of the MSE algorithm for solving mean-payoff games. The algorithm transforms the mean payoff game into an energy game and solves it using an iterative approach with progress measures. Such algorithms are described in [DBLP:journals/iandc/BenerecettiDM24].

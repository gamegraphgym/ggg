Detailed Description

Iterative attractor algorithm for Buchi games.

Solves Buchi games using the classical iterative attractor approach based on [DBLP:conf/stacs/Thomas95]. The algorithm repeatedly computes attractors to accepting vertices until a fixpoint is reached.

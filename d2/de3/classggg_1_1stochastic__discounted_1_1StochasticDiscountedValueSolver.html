Detailed Description

Value iteration algorithm for stochastic discounted games.

Implementation of the classical value iteration method for solving stochastic discounted games based on [DBLP:journals/pnas/Shapley53] and [DBLP:books/wi/Puterman94]. The algorithm iteratively updates value estimates until convergence using Bellman equations with discounting factors.

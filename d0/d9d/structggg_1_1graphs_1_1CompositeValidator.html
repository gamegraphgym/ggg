Detailed Description

template<typename GraphType, Validator< GraphType >... Validators>
struct ggg::graphs::CompositeValidator< GraphType, Validators >

Composite validator that runs multiple validators in sequence.

Combines multiple validators into a single validator that runs all constituent validators in order. Validation fails on the first validator that throws an exception.

Template Parameters
GraphTypeThe graph type to validate
ValidatorsVariadic pack of validator types satisfying Validator concept

Example usage:

using MyValidator = CompositeValidator<
ParityGraph,
PlayerValidator<0, 1>,
PriorityValidator<0>
>;
MyValidator::validate(graph);
